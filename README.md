A simple file based cache engine that implements the Psr cache interfaces

## Configuration

composer.json
<pre>
"require": {
        "micronative/file-cache": "^1.0.0"
},
"repositories": [
    { "type": "vcs", "url": "https://github.com/micronative/file-cache" }
],
</pre>

Run
<pre>
composer require micronative/file-cache:1.0.0
</pre>
